# WTWR (What To Wear)

WTWR is a weather-based web application built with **React** and **Vite** that helps users decide what to wear based on current local weather conditions. The app fetches real-time weather data for the user’s location and displays outfit and garment recommendations tailored to the forecast.

## Features

- Displays current local weather conditions
- Recommends appropriate clothing and garments based on temperature and weather
- Responsive, user-friendly interface
- Fast development and build times powered by Vite

## Tech Stack

- **React** – Component-based UI development
- **Vite** – Modern frontend build tool for fast bundling and hot module replacement
- **JavaScript (ES6+)**
- **CSS** – Styling and layout

## Purpose

WTWR is designed to make daily outfit decisions easier by combining real-time weather data with practical clothing recommendations.
